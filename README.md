# Проект Sprint_5

Автотесты для тестирования функциональности сайта Stellar Burgers.

## Технологии
- Python 3.9+
- Selenium WebDriver
- pytest

## Установка и запуск
1. Клонировать репозиторий
2. Установить зависимости: `pip install selenium pytest`
3. Скачать драйверы браузера (ChromeDriver или GeckoDriver) и поместить в PATH
4. Запустить тесты: `pytest tests --browser=chrome`

## Структура
- `conftest.py` – фикстуры для браузера
- `locators.py` – локаторы элементов
- `generators.py` – генерация уникальных email и паролей
- `tests/` – тесты по группам

## Покрытие
- Регистрация (успешная / ошибка пароля)
- Вход (4 способа)
- Личный кабинет / выход
- Конструктор (переходы по разделам)
